2.2 ¿Por qué realizar evaluaciones de necesidades

En el análisis comercial, se realizan evaluaciones de necesidades para examinar el entorno empresarial y abordar un problema comercial o oportunidad actual. Una evaluación de necesidades puede ser solicitada formalmente por una parte interesada comercial, ordenada por una metodología interna, o recomendada por un analista de negocios antes de iniciar un programa o proyecto. Como se usa en esta guía de práctica, un proyecto es un esfuerzo temporal realizado para crear un producto, servicio o resultado único. Un programa es un grupo de proyectos relacionados, subprogramas y actividades del programa administradas de una manera coordinada para obtener beneficios que no están disponibles al administrarlos individualmente. El trabajo de evaluación de las necesidades se realiza antes de que comience el trabajo del programa o el proyecto, por lo tanto, se dice que involucra las actividades previas al preproyecto. Sin embargo, durante el curso de un proyecto, si los factores externos cambian (por ejemplo, fusión corporativa, gran porcentaje de pérdida de participación de mercado, etc.), que influyen o impactan el proyecto en proceso, el analista de negocios deberá revisar la evaluación y las decisiones de las necesidades Hecho previamente para asegurarse de que todavía sean válidos para la situación que el negocio está abordando.

Una evaluación de necesidades implica la finalización de un análisis de brecha, una técnica descrita en la Sección 2.4.7, que se utiliza para analizar y comparar el rendimiento real de la organización con el rendimiento esperado o deseado. Gran parte del análisis completado durante la evaluación de necesidades se utiliza para el desarrollo de un caso de negocio. Es la evaluación de necesidades y el caso de negocios el que construye las bases para determinar los objetivos del proyecto y servir como aportes para una carta del proyecto.

Cuando se pasa por alto una evaluación de necesidades, a menudo no hay un análisis insuficiente para comprender adecuadamente la necesidad comercial. El analista de negocios realiza una evaluación de necesidades para ayudar a la organización a comprender un problema comercial u oportunidad con mayor detalle para garantizar que se resuelva el problema correcto. Cuando se evita una evaluación de necesidades formales, la solución resultante a menudo no aborda el problema comercial subyacente o no resuelve el problema por completo; También es posible proporcionar una solución que no sea necesaria o una que contenga características innecesarias.
