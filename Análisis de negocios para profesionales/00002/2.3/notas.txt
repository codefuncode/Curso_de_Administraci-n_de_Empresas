2.3 Identificar el problema u oportunidad
Parte del trabajo realizado dentro de la evaluación de necesidades es identificar el problema que se está resolviendo o la oportunidad que debe abordarse. Para evitar concentrarse en la solución demasiado pronto, se hace hincapié en comprender el entorno actual y analizar la información descubierta. El analista de negocios debe preguntar "¿Qué problema estamos resolviendo?" o "¿Qué problemas tienen nuestros clientes que abordarán esta oportunidad?" El analista de negocios comienza a provocar información para descubrir los datos necesarios para identificar completamente el problema u oportunidad.
